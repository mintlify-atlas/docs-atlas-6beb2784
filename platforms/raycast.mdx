# Raycast Extension

The Convex Panel Raycast extension brings powerful Convex debugging tools directly to your macOS launcher. Execute functions, browse data, view logs, and more—all without leaving your keyboard.

![Raycast Extension](https://firebasestorage.googleapis.com/v0/b/relio-217bd.appspot.com/o/convex-panel%2Fconvex%202026-01-15%20at%2009.06.58.png?alt=media&token=074c65f1-812b-4d54-b0b9-b918b3b78f42)

## Installation

<Steps>
  <Step title="Install Raycast">
    Download [Raycast](https://raycast.com) if you haven't already (free for personal use)
  </Step>
  <Step title="Install Extension">
    Visit the [Raycast Store](https://raycast.com/convexpanel/convex-panel) and click "Install Extension"
    
    Or search for "Convex" in Raycast's extension store
  </Step>
  <Step title="Configure Credentials">
    Open Raycast → Extensions → Convex → Configure Extension
    
    Add your deploy key and deployment URL (optional)
  </Step>
  <Step title="Start Using">
    Trigger Raycast (`Cmd + Space`) and start typing a Convex command
  </Step>
</Steps>

## Requirements

- **macOS**: 10.15 (Catalina) or later
- **Raycast**: Latest version recommended
- **Convex**: Active Convex project with API access

## Available Commands

The extension includes 10 powerful commands:

### Manage Projects

Switch between your Convex projects quickly.

```bash
# Command
Manage Projects

# Features
- View all projects in your team
- Switch active project
- See deployment status
- Quick access to dashboard
```

<Note>
  Projects are automatically synced from your Convex account.
</Note>

### Switch Deployment

Toggle between dev, staging, and production deployments.

<Tabs>
  <Tab title="Development">
    Your local development deployment for testing
  </Tab>
  <Tab title="Staging">
    Preview environment for testing before production
  </Tab>
  <Tab title="Production">
    Your live production deployment
  </Tab>
</Tabs>

```typescript
// Command implementation
export default function SwitchDeployment() {
  const deployments = [
    { name: 'Development', url: 'https://dev.convex.cloud' },
    { name: 'Production', url: 'https://prod.convex.cloud' },
  ];
  
  return (
    <List>
      {deployments.map(d => (
        <List.Item key={d.url} title={d.name} />
      ))}
    </List>
  );
}
```

## Extension Preferences

Configure the extension in Raycast settings:

| Preference | Type | Description |
|-----------|------|-------------|
| **Deploy Key** | Password | Convex deploy key for direct access (format: `instance-name\|key`) |
| **Deployment URL** | Text | Your Convex deployment URL (e.g., `https://polite-condor-874.convex.cloud`) |

<CodeGroup>
```json Configuration
{
  "preferences": [
    {
      "name": "deployKey",
      "title": "Deploy Key",
      "description": "Get from Dashboard → Settings → Deploy Key",
      "type": "password",
      "required": false
    },
    {
      "name": "deploymentUrl",
      "title": "Deployment URL",
      "description": "Example: https://polite-condor-874.convex.cloud",
      "type": "textfield",
      "required": false
    }
  ]
}
```
</CodeGroup>

<Warning>
  Keep your deploy key secure! It provides full access to your Convex deployment.
</Warning>

## Extension Architecture

Built with Raycast's modern extension API:

```typescript
// Example command structure
import { List, ActionPanel, Action } from "@raycast/api";
import { useEffect, useState } from "react";

export default function Command() {
  const [items, setItems] = useState([]);
  
  useEffect(() => {
    // Fetch data from Convex API
  }, []);
  
  return (
    <List>
      {items.map(item => (
        <List.Item
          key={item.id}
          title={item.title}
          actions={
            <ActionPanel>
              <Action title="Open" onAction={() => {}} />
            </ActionPanel>
          }
        />
      ))}
    </List>
  );
}
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Extension not showing up">
    - Restart Raycast (Quit and reopen)
    - Check that the extension is enabled in Raycast settings
    - Try reinstalling from the Raycast Store
  </Accordion>
  
  <Accordion title="Authentication errors">
    - Verify your deploy key format: `instance-name|key`
    - Get a fresh deploy key from Convex dashboard
    - Check that your deployment URL is correct
    - Try removing and re-adding credentials
  </Accordion>
  
  <Accordion title="Commands timing out">
    - Check your internet connection
    - Verify the Convex deployment is online
    - Try with a different deployment URL
    - Check Convex dashboard for service status
  </Accordion>
  
  <Accordion title="Can't see my projects">
    - Ensure you have access to the team in Convex dashboard
    - Verify your deploy key has the correct permissions
    - Try the "Manage Projects" command to refresh
  </Accordion>
</AccordionGroup>

## Community Extensions

Build your own Convex commands:

```typescript
// Custom command example
import { showToast, Toast } from "@raycast/api";
import { convexClient } from "./utils/convex";

export default async function Command() {
  try {
    const result = await convexClient.query(
      "myFunction",
      { arg: "value" }
    );
    
    await showToast({
      style: Toast.Style.Success,
      title: "Function executed",
      message: JSON.stringify(result),
    });
  } catch (error) {
    await showToast({
      style: Toast.Style.Failure,
      title: "Error",
      message: error.message,
    });
  }
}
```
