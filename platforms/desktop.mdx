# Desktop App

Convex Panel Desktop is a powerful native application built with Tauri and React that brings comprehensive Convex debugging and monitoring capabilities to your desktop.

![Desktop App Screenshot](https://firebasestorage.googleapis.com/v0/b/relio-217bd.appspot.com/o/convex-panel%2FScreenshot%202026-01-20%20at%2011.23.22%E2%80%AFPM.png?alt=media&token=b34df713-4c3f-45a2-87ff-3f826aa44d3d)

## Installation

<Tabs>
  <Tab title="macOS (Homebrew)">
    ```bash
    brew install robertalv/tap/convex-panel
    ```
  </Tab>
  <Tab title="macOS (curl)">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/robertalv/convex-panel/main/install.sh | bash
    ```
  </Tab>
  <Tab title="Windows (PowerShell)">
    ```powershell
    irm https://raw.githubusercontent.com/robertalv/convex-panel/main/install.ps1 | iex
    ```
  </Tab>
  <Tab title="Manual Download">
    Download the latest release from [GitHub Releases](https://github.com/robertalv/convex-panel/releases):
    - **macOS**: `.dmg` (Apple Silicon & Intel)
    - **Windows**: `.msi` or `.exe` installer
    - **Linux**: `.AppImage` or `.deb`
  </Tab>
</Tabs>

## Configuration

### OAuth Setup

The desktop app uses OAuth for secure authentication:

<Steps>
  <Step title="Open Dashboard">
    Go to your [Convex Dashboard](https://dashboard.convex.dev)
  </Step>
  <Step title="Create OAuth App">
    Navigate to Settings → OAuth and create a new OAuth application
  </Step>
  <Step title="Configure Redirect URI">
    Add `convex-panel://oauth/callback` as a redirect URI
  </Step>
  <Step title="Connect in App">
    Launch the desktop app and click "Connect" to authenticate
  </Step>
</Steps>

### Local Data Storage

The desktop app stores data locally:

- **Logs**: SQLite database for historical log search
- **Preferences**: JSON files for user settings
- **Cache**: Temporary cache for improved performance

<Note>
  All data is stored locally on your machine and never sent to external servers (except Convex API calls).
</Note>

## Building from Source

Want to build the desktop app yourself?

<Steps>
  <Step title="Clone Repository">
    ```bash
    git clone https://github.com/robertalv/convex-panel.git
    cd convex-panel
    ```
  </Step>
  <Step title="Install Dependencies">
    ```bash
    pnpm install
    ```
  </Step>
  <Step title="Build Desktop App">
    ```bash
    cd apps/desktop
    pnpm tauri build
    ```
  </Step>
  <Step title="Find Artifacts">
    Artifacts will be in `apps/desktop/src-tauri/target/release/bundle/`:
    - **macOS**: `macos/*.app`, `dmg/*.dmg`
    - **Windows**: `msi/*.msi`, `nsis/*.exe`
    - **Linux**: `appimage/*.AppImage`, `deb/*.deb`
  </Step>
</Steps>

<Note>
  Building requires Node.js 22+, pnpm 9+, and the Rust toolchain installed.
</Note>

## Troubleshooting

<AccordionGroup>
  <Accordion title="App won't launch on macOS">
    If you see "App is damaged and can't be opened":
    
    ```bash
    xattr -cr /Applications/Convex\ Panel.app
    ```
    
    This removes the quarantine attribute from unverified apps.
  </Accordion>
  
  <Accordion title="Windows SmartScreen warning">
    Click "More info" and then "Run anyway". This appears because the app isn't code-signed yet.
  </Accordion>
  
  <Accordion title="Linux permission issues">
    Make the AppImage executable:
    
    ```bash
    chmod +x convex-panel.AppImage
    ./convex-panel.AppImage
    ```
  </Accordion>
  
  <Accordion title="OAuth not working">
    Ensure you've added `convex-panel://oauth/callback` as a redirect URI in your Convex OAuth settings.
  </Accordion>
</AccordionGroup>
