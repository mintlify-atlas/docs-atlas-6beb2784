## Key Features

### OAuth Device Flow Authentication

Secure authentication without typing passwords on mobile.

<Steps>
  <Step title="Launch App">
    Open the Convex Panel mobile app
  </Step>
  <Step title="Start Device Auth">
    Tap "Connect with Device Code"
  </Step>
  <Step title="Authorize">
    You'll see a verification code and be redirected to authorize in your browser
  </Step>
  <Step title="Complete">
    Once authorized, return to the app—you're connected!
  </Step>
</Steps>

```typescript
// Device authorization flow implementation
const deviceAuth = await authService.startDeviceAuthorization();

// User sees: ABCD-1234
const userCode = deviceAuth.user_code;

// Open browser to verification URL
await WebBrowser.openAuthSessionAsync(
  deviceAuth.verification_uri_complete,
  undefined,
  { preferEphemeralSession: true }
);

// Poll for token
const tokenResponse = await authService.pollForDeviceToken(deviceAuth);
```

<Note>
  Device flow is perfect for mobile apps—no need to type complex passwords or tokens on a small screen.
</Note>

## Building from Source

Want to build the mobile app yourself?

<Steps>
  <Step title="Clone Repository">
    ```bash
    git clone https://github.com/robertalv/convex-panel.git
    cd convex-panel/apps/mobile
    ```
  </Step>
  
  <Step title="Install Dependencies">
    ```bash
    npm install
    ```
  </Step>
  
  <Step title="Start Development Server">
    ```bash
    npx expo start
    ```
  </Step>
  
  <Step title="Run on Device or Emulator">
    - **iOS**: Press `i` to open in iOS Simulator
    - **Android**: Press `a` to open in Android Emulator
    - **Device**: Scan QR code with Expo Go app
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Device auth not working">
    - Ensure you have an active internet connection
    - Try clearing the app cache in settings
    - Verify your Convex OAuth settings allow device flow
    - Check that you're using the latest app version
  </Accordion>
  
  <Accordion title="Push notifications not received">
    - Enable notifications in device Settings → Convex Panel
    - Check notification settings within the app
    - Verify notification permissions were granted
    - Test notifications from the Account screen
  </Accordion>
  
  <Accordion title="App crashes on launch">
    - Try uninstalling and reinstalling the app
    - Check for available app updates
    - Report the issue with device info to GitHub issues
  </Accordion>
  
  <Accordion title="Can't see my deployments">
    - Verify you're authenticated (check Account screen)
    - Ensure you have access to the team in Convex dashboard
    - Try logging out and back in
    - Pull down to refresh the deployment list
  </Accordion>
</AccordionGroup>
