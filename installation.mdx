Convex Panel is available across multiple platforms to fit your workflow. Choose the installation method that works best for you.

## Desktop App

The desktop app provides a standalone application for macOS, Windows, and Linux.

### macOS

<Tabs>
  <Tab title="Homebrew">
    ```bash
    brew install robertalv/tap/convex-panel
    ```
  </Tab>
  
  <Tab title="curl">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/robertalv/convex-panel/main/install.sh | bash
    ```
  </Tab>
  
  <Tab title="Manual Download">
    Download the latest `.dmg` file from [GitHub Releases](https://github.com/robertalv/convex-panel/releases).
    
    Available for both Apple Silicon and Intel Macs.
  </Tab>
</Tabs>

### Windows

<Tabs>
  <Tab title="PowerShell">
    ```powershell
    irm https://raw.githubusercontent.com/robertalv/convex-panel/main/install.ps1 | iex
    ```
  </Tab>
  
  <Tab title="Manual Download">
    Download the latest installer from [GitHub Releases](https://github.com/robertalv/convex-panel/releases):
    - `.msi` installer (recommended)
    - `.exe` installer (alternative)
  </Tab>
</Tabs>

### Linux

Download from [GitHub Releases](https://github.com/robertalv/convex-panel/releases):

- **AppImage**: Universal Linux application (no installation required)
- **deb**: For Debian/Ubuntu-based distributions

```bash
# For .deb packages
sudo dpkg -i convex-panel_*.deb

# For AppImage
chmod +x Convex-Panel-*.AppImage
./Convex-Panel-*.AppImage
```

## NPM Package

Embed Convex Panel directly in your React application for development.

<Steps>
  <Step title="Install the package">
    <CodeGroup>
      ```bash npm
      npm install convex-panel --save-dev
      ```
      
      ```bash yarn
      yarn add convex-panel --dev
      ```
      
      ```bash pnpm
      pnpm add convex-panel --save-dev
      ```
      
      ```bash bun
      bun add convex-panel --dev
      ```
    </CodeGroup>
  </Step>
  
  <Step title="Run the setup wizard">
    ```bash
    npx convex-panel setup
    ```
    
    This interactive command will:
    - Create or update `convex/http.ts` with OAuth endpoints
    - Guide you through configuring environment variables
    - Set up OAuth authentication for your project
  </Step>
  
  <Step title="Add to your application">
    See the [Quickstart](/quickstart) guide for framework-specific integration instructions.
  </Step>
</Steps>

<Note>
  The NPM package requires Node.js 18+ and React 18+. See [Requirements](#requirements) for full details.
</Note>

## Raycast Extension

Access Convex Panel directly from your macOS launcher.

### Installation

Install from the [Raycast Store](https://raycast.com/convexpanel/convex-panel)

### Available Commands

| Command | Description |
|---------|-------------|
| Manage Projects | Switch between Convex projects |
| Switch Deployment | Toggle between dev/staging/prod |
| Run Function | Execute Convex functions directly |
| Data Browser | Quick access to table data |
| View Logs | Stream logs in Raycast |
| Search Docs | Search Convex documentation |
| Browse Components | Explore the component registry |
| Open Dashboard | Jump to Convex dashboard |
| Copy Deployment URL | Copy your deployment URL |
| Configure Deploy Key | Set up deploy key authentication |

<Note>
  The Raycast extension requires macOS 10.15+ and Raycast 1.50+
</Note>

## Verification

### Desktop App

After installation, verify by opening the app:

```bash
# macOS
open "/Applications/Convex Panel.app"

# Windows (PowerShell)
Start-Process "Convex Panel"

# Linux (if installed via deb)
convex-panel
```

### NPM Package

Verify the installation:

```bash
npm list convex-panel
```

Check that the CLI is available:

```bash
npx convex-panel --version
```
