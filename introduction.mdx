## What is Convex Panel?

Convex Panel is a comprehensive debugging and monitoring suite for [Convex](https://convex.dev) applications. It provides developers with powerful tools to inspect data, view logs, run functions, and monitor performance in real-time.

## Core Features

### Data Browser

Browse and manage your Convex database with a powerful data explorer:

- **Multiple view modes**: Table, list, JSON, and raw views
- **In-place editing**: Double-click to edit documents directly
- **Advanced filtering**: Query-based filtering with date ranges
- **Bulk operations**: Export and manage multiple documents at once

### Functions & Logs

Debug and monitor your Convex functions in real-time:

- **Function browser**: Browse all queries, mutations, and actions
- **Source code viewer**: View function source with syntax highlighting
- **Live log streaming**: Monitor function calls as they happen
- **Historical search**: Search through past logs with SQLite storage
- **Call trees**: Trace function execution flow

### Health Metrics

Monitor your application's health at a glance:

- **Failure rate tracking**: Monitor error rates and success metrics
- **Cache hit rate**: Track caching efficiency
- **Latency monitoring**: View response times with percentile breakdowns
- **Database usage**: Monitor storage and document counts

### Schema Visualizer

Visualize your database schema with an interactive diagram:

- **Interactive diagram**: ReactFlow-based schema visualization
- **Git integration**: Visual diff mode for schema changes
- **Multiple layouts**: Force-directed, hierarchical, and radial layouts
- **Export options**: Save diagrams as PNG or SVG

## Who is Convex Panel For?

Convex Panel is designed for developers building applications with Convex:

- **Full-stack developers** who need to debug both frontend and backend issues
- **Backend developers** monitoring function performance and database health
- **DevOps engineers** tracking production deployments and system health
- **Mobile developers** who need remote access to debugging tools
- **Solo developers** who want a comprehensive debugging suite without complex setup
